# Law-Judge 民商事司法分析系统

基于多智能体协作与混合检索技术的法律决策支持系统。系统专注于复杂民商事案件的事实梳理、证据分析、法律适用推演及模拟裁判。

---

## 核心功能

### 1. 案情梳理与要素提取 (Phase 1)
- **案由自动识别**：支持识别股权转让、公司决议、商事合同等核心民商事案由。
- **关键要素锁定**：基于请求权基础理论，协助用户查明案件构成要件（如支付凭证、工商登记等）。
- **对抗性事实检验**：模拟对抗视角，参考历史判例的辩论逻辑，对用户主张的完整性进行评估。

### 2. 模拟法庭与裁判推演 (Phase 2)
- **多智能体博弈执行**：由原告代理人、被告代理人及中立裁判官三个智能体进行逻辑推演。
- **IRAC 推理逻辑**：遵循“争点(Issue)-规则(Rule)-适用(Application)-结论(Conclusion)”的法律实务逻辑。
- **自动化文书起草**：根据推演结果，辅助生成符合规范要求的民事裁判文书初稿。

### 3. 文档处理与混合检索
- **混合检索方案**：整合向量数据库 (ChromaDB) 与全文本检索引擎 (Whoosh)，实现法律名词与语义的精准匹配。
- **结构化文档入库**：支持 Word 格式法律文书。法条采用行首正则精准切分，案例采用 LLM 语义结构化提炼。
- **幂等性保障**：系统支持重复上传。相同内容会自动更新，不会产生冗余数据。

## 运行指南

### 环境依赖
- **Python**: 3.10+
- **Node.js**: v18+

### 1. 环境初始化 (Windows)
在项目根目录运行初始化脚本：
```bash
setup.bat
```
该脚本将自动配置 Python 虚拟环境、安装依赖包并初始化前端环境。

### 2. 配置参数
编辑根目录下的 `.env` 文件，完善相关配置：
```env
DEEPSEEK_API_KEY=您的API_KEY
SILICONFLOW_API_KEY=您的API_KEY
SEEKDB_PATH=./data/seekdb
```

### 3. 启动服务
- **启动后端服务**：运行 `run_backend.bat` (默认端口: 8001)
- **启动前端界面**：运行 `run_frontend.bat`

## 技术架构
- **流程编排**: LangGraph
- **后端框架**: FastAPI
- **数据存储**: ChromaDB + Whoosh
- **前端技术**: React + Vite + Framer Motion
