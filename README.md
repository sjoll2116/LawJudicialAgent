# 民商事司法分析系统

基于多智能体协作与混合检索技术的法律决策支持系统。系统专注于复杂民商事案件的事实梳理、证据分析、法律适用推演及模拟裁判。

---

## 核心功能

### 1. 智能意图识别与双路径处理
系统通过 **Reception Agent** 实时诊断用户意图，实现差异化响应路径：
-   **快速咨询路径**：针对定性法律问题（如：合同是否生效？），直接检索关联法条并给出专家级、确定的法律结论，严禁模棱两可的回答。
-   **复杂案情路径**：针对存在争议、需要证据闭环的案件，自动启动多阶段智能体协作（事实查明 -> 证据质证 -> 法律推演）。


### 2. 模拟法庭与裁判推演 
- **多智能体博弈执行**：由原告代理人、被告代理人及中立裁判官三个智能体进行逻辑推演。
- **IRAC 推理逻辑**：遵循“争点-规则-适用-结论”的法律实务逻辑。

### 3. 文档处理与混合检索
- **混合检索方案**：整合向量数据库 (ChromaDB) 与全文本检索引擎 (Whoosh)，实现多路召回。
- **结构化文档入库**：基于llamaindex接入器支持pdf、docx、txt格式。法条采用行首正则精准切分，案例采用 LLM 语义结构化提炼。自动提取元数据，便于实现元数据过滤。
- **幂等性保障**：系统支持重复上传。相同内容会自动更新，不会产生冗余数据。

## 运行指南

### 环境依赖
- **Python**: 3.10+
- **Node.js**: v18+

### 1. 环境初始化 (Windows)
在项目根目录运行初始化脚本：
```bash
setup.bat
```
该脚本将自动配置 Python 虚拟环境、安装依赖包并初始化前端环境。

### 2. 配置参数
编辑根目录下的 `.env` 文件，完善相关配置：
```env
DEEPSEEK_API_KEY=您的API_KEY
SILICONFLOW_API_KEY=您的API_KEY
SEEKDB_PATH=./data/seekdb
```

### 3. 启动服务
- **启动后端服务**：运行 `run_backend.bat` (默认端口: 8001)
- **启动前端界面**：运行 `run_frontend.bat`
